# CodeAlpha Random Quote Generator 💬

A minimal and elegant Flutter application that displays random inspirational quotes with smooth animations. This is my **second task** for the **App Development Internship at CodeAlpha**.

## ✨ Features
* **Instant Quotes:** Get a new inspirational quote with every click.
* **Smooth Animations:** Features a 3D-like scale and fade transition using `AnimatedSwitcher`.
* **Copy to Clipboard:** Easily copy quotes to share with friends.
* **Premium UI:** Deep indigo gradient design with clean typography.
